Cookie Consent System

GDPR-compliant cookie management with 10-day persistence and enhanced user experience

System Overview

Our centralized cookie consent system provides a comprehensive solution for managing user privacy preferences across the entire website. The system is designed to be GDPR-compliant while maintaining an excellent user experience.

GDPR Compliance

  • Clear consent options
  • Easy withdrawal of consent
  • Detailed privacy policy links
  • Granular control options

Technical Features

  • 10-day consent persistence
  • LocalStorage management
  • Google Analytics integration
  • Custom event dispatching

User Experience

  • Responsive design
  • Smooth animations
  • Accessibility support
  • Mobile-optimized

Implementation

The cookie consent system is implemented as a JavaScript class that can be easily integrated into any page. Simply include the script and the system will automatically handle consent management.

<!-- Include the cookie consent script --> <script src="../js/cookie-consent.js"></script> <!-- The system initializes automatically --> <script> // Optional: Listen for consent changes window.addEventListener('cookieConsentChanged', function(event) { console.log('Consent changed:', event.detail); }); </script>

Integration Guide

To integrate this cookie consent system into other pages:

1. Include the script: <script src="js/cookie-consent.js"></script> 2. The system will automatically initialize when the page loads 3. Consent is stored for 10 days in localStorage 4. Listen for 'cookieConsentChanged' events for custom handling 5. Use window.coralCookieConsent.getConsentStatus() to check current status
