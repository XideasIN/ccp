Centralized Footer Integration Example

How to Use the Centralized Footer

The centralized footer component has been created as footer.html. Here's how to integrate it into any page:

Method 1: jQuery Load (Recommended)

<!-- Add this div where you want the footer to appear --> <div id="footer-placeholder"></div> <!-- Include jQuery and load the footer --> <script src="https://code.jquery.com/jquery-3.6.0.min.js"></script> <script> $(document).ready(function() { $("#footer-placeholder").load("footer.html"); }); </script>

Method 2: Server-Side Include (PHP)

<?php include 'footer.html'; ?>

Method 3: JavaScript Fetch

<div id="footer-placeholder"></div> <script> fetch('footer.html') .then(response => response.text()) .then(data => { document.getElementById('footer-placeholder').innerHTML = data; }); </script>
Important: The footer includes automatic path resolution JavaScript that adjusts all links and images based on the current page's location in the directory structure. This ensures all links work correctly regardless of where the page is located.
