Step 1: Add the Container

Add this HTML container where you want the footer to appear (typically before the closing </body> tag):

Step 2: Include the Script

Add the footer script before the closing </body> tag, after the container:

üîß Path Adjustment for Subdirectories

Automatic Path Resolution: The footer script automatically detects the current page's directory depth and adjusts all paths accordingly. No manual configuration needed!

How it works:

  • Root level (index.html): Uses direct paths (e.g., "wp-content/logo/Logo_white.png")
  • One level deep (about-us/index.html): Uses "../" prefix (e.g., "../wp-content/logo/Logo_white.png")
  • Two levels deep (sectors/construction/index.html): Uses "../../" prefix
  • And so on... Automatically calculates the correct number of "../" prefixes
